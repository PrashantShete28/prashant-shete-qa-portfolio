Project 1 – Loan Origination System (LOS / LMS – Banking)

Current Project

This project is a digital banking platform used for applying, processing, and managing loans across their complete life cycle. I worked extensively on both LOS and LMS modules, ensuring accurate data flow, validation, and compliance with banking standards.

  • Worked on LOS modules including QDE (Quick Data Entry) and DDE (Detailed Data Entry)
  • Validated customer onboarding, KYC verification, and risk assessment workflows

  • Verified EMI calculation logic, repayment schedules, and interest handling
  • Performed end-to-end testing of loan disbursement and repayment flows in LMS
  • Conducted API testing and backend SQL validation for loan stage data
